# Infer Demo Tool

## Overview

Infer is a static analysis tool for detecting bugs in code without running it. This demo shows how it can be used during development.

## What We Did

* Wrote a script to run Infer
* The script lets users choose actions (1–5), such as:

  * Analyze current branch
  * Compare with `main`
  * View reports

## Testing

* Created a new branch
* Added bugs to the code

This simulates real development scenarios.

## Result

Infer can:

* Detect bugs automatically
* Generate reports for developers

## Conclusion

Infer helps find issues early and supports the development process.
